# 05. 디자인 — 명랑 헬시

## 디자인 철학

건강은 즐거운 것. 컬러와 이모지로 응원하고, 둥근 모서리와 따뜻한 그라데이션으로 친근함을 준다.

## 컬러 토큰

| 토큰 | 값 | 용도 |
|---|---|---|
| `--bg-primary` | `#fffdf7` | 앱 배경 |
| `--green-main` | `#66BB6A` | 긍정 지표, 남은 칼로리, 추천 |
| `--green-dark` | `#4CAF50` | 강조 숫자, 섹션 타이틀 |
| `--orange-main` | `#FFA726` | 목표, 경고 전 단계, 포인트 |
| `--orange-dark` | `#FF9800` | 헤더 포인트 |
| `--red-warn` | `#EF5350` | 초과 경고, 나쁜 음식 태그 |
| `--text-primary` | `#2d2d2d` | 본문 텍스트 |
| `--text-secondary` | `#5d4e37` | 코멘트, 보조 텍스트 |
| `--text-muted` | `#bba` | 라벨, 단위 |
| `--card-bg` | `#ffffff` | 카드 배경 |
| `--card-shadow` | `0 2px 8px rgba(0,0,0,0.04)` | 카드 그림자 |
| `--gradient-warm` | `linear-gradient(135deg, #FFF8E1, #F1F8E9)` | 코멘트 카드 배경 |
| `--gradient-green` | `linear-gradient(135deg, #E8F5E9, #F1F8E9)` | 메인 카드 배경 |
| `--gradient-chip` | `linear-gradient(135deg, #E8F5E9, #FFF8E1)` | 칩 배경 |
| `--gradient-btn` | `linear-gradient(135deg, #66BB6A, #FFA726)` | 추가 버튼 |

## 타이포그래피

| 요소 | 크기 | 굵기 | 색상 |
|---|---|---|---|
| 헤더 타이틀 | 18px | 700 | `--text-primary`, 포인트 `--orange-dark` |
| 남은 칼로리 숫자 | 44px | 800 | `--green-dark` |
| 섹션 타이틀 | 13px | 700 | `--green-dark` 또는 `--orange-dark` |
| 카드 수치 | 22px | 700 | `--green-dark` 또는 `--orange-main` |
| 본문/음식명 | 14px | 400 | `--text-primary` |
| 라벨/단위 | 11~12px | 400 | `--text-muted` |

## 컴포넌트 명세

### 1. 한 줄 코멘트 카드
- 배경: `--gradient-warm`
- 둥근 모서리: 16px
- 이모지 + 자연어 메시지
- 상태별 이모지: 여유(💪), 주의(⚡), 초과(😅)

### 2. 메인 칼로리 카드
- 배경: `--gradient-green`
- 둥근 모서리: 20px
- 중앙: 원형 링 차트 (160x160px, SVG)
  - 트랙: `#e0e0d0`, 10px
  - 채우기: 그린→오렌지 그라데이션, 10px, round cap
  - 중앙 텍스트: 남은 칼로리 숫자 + "kcal 남음"
- 하단: "860 / 1,650 kcal 섭취" 텍스트
- 프로그레스 바 없음 (원형 링이 시각화 담당)

### 3. 통계 카드 (2열 그리드)
- 배경: `--card-bg`, 그림자 적용
- 둥근 모서리: 16px
- 이모지 아이콘 + 라벨 + 수치
- 목표(🎯): 오렌지, 섭취(🍽️): 그린

### 4. 빠른 추가 칩
- 배경: `--gradient-chip`
- 둥근 모서리: 20px (pill)
- 이모지 + 음식명 + 칼로리
- 가로 스크롤/플렉스 래핑

### 5. 오늘 먹은 것 리스트
- 카드 컨테이너 안에 리스트
- 각 항목: 건강 태그 dot(8px) + 음식명 + 칼로리
- 태그 색상: 좋음(🟢 `#66BB6A`), 주의(🔴 `#EF5350`), 보통(⚪ `#ddd`)
- 구분선: `#f5f0e8`

### 6. 추천 음식 섹션
- 카드 컨테이너
- 각 항목: 그린 배경 카드 (`--gradient-chip`), 12px 둥근 모서리
- 건강 태그 dot + 음식명 + 칼로리

### 7. 추가 버튼 (FAB)
- 60x60px 원형
- 배경: `--gradient-btn`
- 그림자: `0 4px 16px rgba(102,187,106,0.4)`
- 하단 중앙 고정

## 레이아웃 구조

```
[상태바]
[헤더: "오늘의 칼로리"]
[한 줄 코멘트 카드]
[메인 칼로리 카드 - 원형 링]
[통계 카드 2열: 목표 | 섭취]
[빠른 추가 칩]
[오늘 먹은 것 리스트]
[추천 음식 섹션]
[추가 버튼 FAB]
```

- 단일 컬럼 스크롤 (모바일)
- 카드 간격: 12px
- 좌우 패딩: 16px

## 반응형

| 브레이크포인트 | 변경 |
|---|---|
| ~480px (모바일) | 기본 레이아웃, 단일 컬럼 |
| 481~768px (태블릿) | 카드 패딩 증가, 원형 링 200px |
| 769px~ (데스크톱) | 최대 너비 480px 중앙 정렬, 좌우 여백 |

## 인터랙션

| 동작 | 효과 |
|---|---|
| 칩/카드 탭 | scale(0.97) + 0.1s ease |
| 음식 추가 시 | 원형 링 애니메이션 (stroke-dashoffset 트랜지션 0.5s) |
| 칼로리 초과 시 | 원형 링 색상 빨간색 전환, 코멘트 이모지 변경 |
| 음식 삭제 | 좌측 스와이프 또는 삭제 버튼 → fade out |
| FAB 탭 | 음식 입력 모달/시트 올라옴 |

## 접근성

- 색상 대비: WCAG AA 기준 충족 (본문 4.5:1 이상)
- 터치 타깃: 최소 44x44px
- 원형 링: aria-label로 "남은 칼로리 790kcal, 전체 1650kcal 중 52% 섭취" 제공
- 건강 태그: 색상 외에도 aria-label로 "좋은 음식"/"주의 음식" 표시
